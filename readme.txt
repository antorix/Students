УСТАНОВКА

Students не требует установки, но необходимо установить движок Python. Для этой цели в Windows просто запустите файл УСТАНОВКА.exe. Проверьте соединение с Интернетом – будет загружено примерно 13 МБ. Сама программа запускается файлом Students.pyw. Можно создать ярлык на рабочем столе с помощью прилагаемой иконки (файл icon.ico).

Если при запуске файла Halieus.pyw в Windows возникает ошибка типа «Отсутствует файл api-ms-win-crt-runtime-l1-1-0.dl», см. здесь: https://helpx.adobe.com/ru/creative-cloud/kb/error_on_launch.html

Ubuntu Linux

Введите:
sudo apt-get install idle3

Затем откройте файл Students.pyw через IDLE3 и нажмите F5.

MAC

В OS X установите Python по ссылке:
https://www.python.org/ftp/python/3.6.0/python-3.6.0-macosx10.6.pkg

Затем необходимо установить TCL по ссылке:
http://www.activestate.com/activetcl/downloads/thank-you?dl=http://downloads.activestate.com/ActiveTcl/releases/8.5.18.0/ActiveTcl8.5.18.0.298892-macosx10.5-i386-x86_64-threaded.dmg 

Затем откройте файл Students.pyw через IDLE3 и нажмите F5. В некоторых случаях возникающие проблемы решаются за счет запуска в режиме администратора.

РАБОТА

После выполнения студентами учебных заданий обновите данные каждого из них, проставив соответствующие даты, заметки, уроки и советы (при необходимости). Затем можно легко отсортировать список по различным параметрам, чтобы назначить новые задания. Наиболее удобно сортировать по датам, чтобы быстро увидеть, кто давно не выступал и не помогал как партнер. В поле урока могут быть не только цифры, но и буквы, при этом сортировка осуществляется только по числу. В карточках есть пара дополнительных полей: «Совет» (какой совет получил последний раз) и «Все пройденные уроки» (сюда можно вписывать историю уроков, которые прошел учащийся). Оба поля можно использовать по желанию, но они могут быть полезны. По кнопке «НХЖС» можно открыть рабочую тетрадь с официального сайта, чтобы узнать задания. Конкретный адрес страницы можно изменить в настройках. Кнопка «Задания» открывает произвольный PDF-файл – предполагается, что это файл бланков, куда вы будете вписывать задания и затем распечатывать их. Поместите этот файл в папку программы и пропишите его название в настройках. Чтобы быстро вписать в бланк имя студента, просто щелкните по нему мышкой в списке, его имя скопируется в буфер обмена, и затем вставьте имя в бланк по Ctrl-V. В настройках можно ввести заметку, которая будет отображаться на главной странице.

ДАННЫЕ И РЕЗЕРВИРОВАНИЕ

Данные всех учащихся хранятся в обычном текстовом txt-файле «students.txt», который можно резервировать в облако и, в общем-то, делать с ним все что угодно. Файл можно просматривать, но не стоит редактировать его вручную без острой необходимости. Любое, даже незначительное изменение форматирования может превратить вашу библиотеку в хаос! При проблемах с файлом students.txt можно восстановить его из одной из последних резервных копий (автоматически создаются в папке /backup после каждой модификации файла). Восстановление делается функцией «Импорт» в меню «Файл». Таким же способом можно и просто загрузить стороннюю базу данных.

ОБНОВЛЕНИЕ

Начиная с версии 3.0.3 программа обновляется автоматически в фоновом режиме. Размер загружаемого файла минимальный (считанные килобайты), поэтому проверка и установка обновления никак не препятствуют работе. Но если у вас более старая версия, для обновления нужно вручную загрузить архив программы и перезаписать все существующие файлы. Переустанавливать Python при этом не нужно.
